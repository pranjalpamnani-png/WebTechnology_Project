## Project Description
MiniHelpDesk is a full-stack helpdesk ticket management system. Users can create support tickets, view all tickets, delete tickets, and see a live dashboard showing ticket statistics.

## How to Install

### Backend
```bash
cd backend
npm install
```

### Frontend
```bash
npm install
```

---

## Environment Variables
Create a `.env` file inside the `backend/` folder:
```
MONGO_URI=mongodb://localhost:27017/minihelpdesk
PORT=5000
```

---

## MongoDB Setup
1. Download and install [MongoDB Community Server](https://www.mongodb.com/try/download/community)
2. MongoDB will run automatically as a Windows service
3. The `minihelpdesk` database is created automatically when the first ticket is created
4. You can view data using MongoDB Compass at `mongodb://localhost:27017`

---

## How to Run

### Run Backend
```bash
cd backend
npx ts-node --transpile-only src/server.ts
```
Backend runs on: `http://localhost:5000`

### Run Frontend
```bash
npm run dev
```
Frontend runs on: `http://localhost:5173`

> Both terminals must be open at the same time

## API Endpoints
| Method | Route | Description |
|--------|-------|-------------|
| GET | /tickets | Get all tickets |
| POST | /tickets | Create a new ticket |
| DELETE | /tickets/:id | Delete a ticket by ID |
| GET | /tickets/stats | Get dashboard statistics |

---

## Ticket Model
| Field | Type | Values |
|-------|------|--------|
| subject | String | Any text |
| description | String | Any text |
| priority | String | Low, Medium, High |
| status | String | Open, In Progress, Closed |
| createdAt | Date | Auto generated by MongoDB |

---

## Implemented Features

### Product Feature вЂ” Dashboard Summary
- Backend endpoint `GET /tickets/stats` queries MongoDB using `countDocuments()`
- Returns total tickets, open, in progress, closed, and high priority counts
- Frontend `Dashboard.tsx` fetches stats using `useEffect` and displays 5 stat cards

### Engineering Feature вЂ” Not Found Page
- `NotFound.tsx` component shows a styled 404 page
- In `App.tsx` the route `path="*"` catches any unknown URL
- Includes a Go Back Home button using React Router `Link`
